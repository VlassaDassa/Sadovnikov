TYPOGRAPHY

Файл: typography.scss
Управление типографикой: шрифт, начертание, размер, высота строки. Там же есть миксин, который будет применять как "на месте", так и в компонентах (components/general)
Организация: компоненты будут созданы для наиболее повторящихся элементов: заголовки, просто абзацный текст
Управление размером: в общем файле index.scss в тэге html, поскольку размеры заданы в rem

P.S
У меня к сожалению, очень странный дизайн, где мало одинаковых элементов, даже заголовки прыгают. Таков стиль. Но поскольку я делаю проект итеративно, т.е сейчас из дизайна у меня сделана только главная страница в трёх размерах, а я уже делаю код, то вероятно, остальные страницы будут более унифицированы и что-то поменяется.
